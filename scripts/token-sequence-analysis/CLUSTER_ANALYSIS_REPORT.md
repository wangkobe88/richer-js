## 关键发现

### 1. "极简暴利"模式 (聚类 2)
- 只有 **2 笔交易**
- 净流入高达 **$11,800**
- 涨幅中位数 **163.5%**
- **可能的模式**: 大户买入 → 价格上涨 → 大户卖出，散户来不及参与

### 2. "超高活跃优质币" (聚类 0)
- 交易极其活跃 (最多 2090 笔)
- 钱包参与度极高 (最多 683 个)
- 涨幅表现优异 (中位数 317.8%)
- **可能的模式**: 广泛参与、持续关注的优质项目

### 3. 早期爆发模式 (聚类 1)
- 序列短 (中位数 35 笔)
- 但涨幅高 (中位数 190.2%)
- **可能的模式**: 早期快速上涨，后期缺乏持续参与

## 交易策略建议

### 建议关注的模式

1. **聚类 0 (超高活跃)**: 可持续增长的优质币
   - 特征: 序列长度 > 500, 唯一钱包 > 200
   - 策略: 长线持有，关注社区活跃度

2. **聚类 1 (早期爆发)**: 短期机会
   - 特征: 序列长度 < 100, 但净流入 > $1000
   - 策略: 快进快出，把握早期机会

### 需要警惕的模式

1. **聚类 2 (极简暴利)**: 大户控盘
   - 特征: 序列长度 < 5, 净流入异常高
   - 风险: 散户难以参与，容易被割

2. **聚类 3 (普通模式)**: 表现平平
   - 特征: 各项指标中等
   - 建议: 谨慎参与

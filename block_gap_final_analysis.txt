六、新增因子定义
================================================================================

建议添加以下因子到 EarlyParticipantCheckService:

1. blockGapStdDev (标准差)
   - 定义: 区块号间隔的标准差
   - 计算: sqrt(sum((x - mean)^2) / n)
   - 预期: 拉砸 < 2.0, 正常 >= 2.0

2. blockGapP75 (75分位数)
   - 定义: 区块号间隔的75分位数
   - 预期: 拉砸 < 3, 正常 >= 3

3. gap0Ratio (gap=0比例)
   - 定义: 同一区块交易的占比
   - 计算: gap=0数量 / 总间隔数
   - 预期: 拉砸 > 0.30, 正常 < 0.30

4. concentration (集中度)
   - 定义: gap<=1的占比（相邻或同区块）
   - 计算: (gap=0 + gap=1)数量 / 总间隔数
   - 预期: 拉砸 > 0.50, 正常 < 0.50

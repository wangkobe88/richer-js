# 交易策略文档

## 一、策略概述

本策略基于对 6,884 个代币、113,892 条 K 线数据的深度分析，针对 four.meme 平台新代币的自动化交易策略。

## 二、核心发现

### 2.1 买入时间点分析

经过对不同买入时间点（1/3/5/10/15分钟）的对比测试，**1分钟买入明显最优**：

| 买入时间 | 80-120%区间中位数 | 盈利率≥50% | 亏损率 |
|---------|------------------|-----------|--------|
| **1分钟** | **47.4%** | **47.1%** | 42.1% |
| 3分钟 | -7.3% | 37.1% | 52.0% |
| 5分钟 | -14.0% | 31.5% | 53.5% |
| 10分钟 | 4.3% | 31.8% | 47.5% |
| 15分钟 | 4.2% | 21.0% | 44.5% |

### 2.2 earlyReturn 区间分析

| earlyReturn 区间 | 样本数 | 覆盖比例 | 中位数收益 | 盈利率≥50% |
|-----------------|-------|---------|-----------|-----------|
| **80-120%** | 461 | 7.6% | **47.4%** | **47.1%** |
| 100-120% | 199 | 3.3% | 50.7% | 50.8% |
| 60-120% | 780 | 12.8% | 45.2% | 45.0% |
| 40-150% | 1383 | 22.7% | 43.7% | 43.3% |

**推荐区间：80-120%**
- 覆盖约 7.6% 的代币，筛选效果强
- 样本量充足（461个）
- 中位数收益 47.4%

## 三、交易策略

### 3.1 买入条件

**触发时间：代币创建后 1 分钟**

**必要条件：**
- earlyReturn 在 80-120% 之间

**earlyReturn 计算公式：**
```
earlyReturn = (1分钟时收盘价 - 第1根K线开盘价) / 第1根K线开盘价 × 100%
```

### 3.2 卖出策略（阶梯止盈）

| 触发条件 | 操作 | 说明 |
|---------|------|------|
| 当前收益 ≥ +30% | 卖出 50% 仓位 | 第一次止盈 |
| 当前收益 ≥ +50% | 卖出剩余 50% 仓位 | 第二次止盈，全部退出 |
| 5分钟内未达到+30% | 持有到观察窗口结束 | 观望策略 |
| 30分钟观察窗口结束 | 退出 | 强制结束 |

### 3.3 买入后流程图

```
买入 (1分钟时, earlyReturn 80-120%)
    │
    ├── 每分钟检查当前收益
    │     │
    │     ├── 收益 ≥ +30%? → 卖出50% → 继续监控
    │     │                            │
    │     │                            └── 收益 ≥ +50%? → 卖出剩余 → 完成
    │     │
    │     ├── 5分钟已过且未达+30%? → 持有到30分钟 → 完成
    │     │
    │     └── 30分钟已过? → 强制退出 → 完成
    │
    └── 完成交易，移出监控池
```

## 四、预期表现

基于历史数据回测（80-120% 区间，1分钟买入）：

| 指标 | 数值 |
|-----|------|
| 样本数 | 461 |
| 平均收益 | 26.2% |
| 中位数收益 | 47.4% |
| 盈利率≥50% | 47.3% |
| 翻倍率≥100% | 13.7% |
| 亏损率 | 42.1% |

### 收益分布

- 亏损 (<0%): 42.1%
- 小赚 (0-30%): 5.8%
- 中赚 (30-50%): 11.8%
- 大赚 (50-100%): 37.1%
- 翻倍 (≥100%): 13.7%

### 风险提示

1. **约 42% 的交易会亏损** - 这不是"稳赚"策略
2. **中位数收益 47.4%** - 这是典型情况，不是保证
3. **极端情况** - 有可能遇到接近 100% 亏损的情况
4. **样本偏差** - 数据仅包含被 AVE API 收集的代币（幸存者偏差）

## 五、风险控制

1. **仓位控制**: 单笔交易不超过总资金的 2-3%
2. **止损纪律**: 单日亏损超过 10% 停止交易
3. **分散投资**: 不要把所有资金投入单一策略
4. **持续监控**: 定期回顾交易结果，必要时调整策略

## 六、实施要点

1. **只操作 four.meme 平台的新代币**
2. **严格在 1 分钟时判断**，不早不晚
3. **earlyReturn 必须在 80-120% 区间**，不放宽标准
4. **严格执行阶梯止盈**，不贪婪不恐惧
5. **超过 30 分钟强制退出**，不恋战

## 七、策略局限性

1. 只适用于 four.meme 平台的 BSC 链代币
2. 依赖于 AVE API 的数据准确性和及时性
3. 策略基于历史数据，未来市场可能变化
4. 滑点、手续费等实际交易成本未计算在内

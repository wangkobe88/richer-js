📖 叙事分析

当因子满足比例达到此值时，触发叙事分析任务（0-100）
虚拟/实盘模式等待叙事分析完成的最长时间（秒）
轮询叙事分析结果的间隔时间（毫秒）
💡

说明

  • 独立引擎异步执行：叙事分析由独立引擎处理，不阻塞实验引擎主循环
  • 分阶段输出：Stage 1 完成时 narrativeRating=8（中间值），Stage 2 完成时返回最终评级 1/2/3
  • 轮询等待机制：实验引擎会轮询等待分析结果（虚拟/实盘默认10秒，回测5分钟）
  • 使用建议：使用条件 narrativeRating > 1 确保只购买已评级的代币，或 narrativeRating >= 2 只要评分完成就购买
  • 优先级：因子满足比例越高，任务优先级越高，优先被分析
  • 缓存复用：同一代币只分析一次，结果跨实验共享

可用因子

因子 类型 说明 示例
age 数字 代币年龄（分钟） age < 1.33
earlyReturn 数字 早期收益率（%） earlyReturn >= 80
currentPrice 数字 当前价格 currentPrice > 0
collectionPrice 数字 收集时的基准价格 currentPrice > collectionPrice
highestPrice 数字 历史最高价格 currentPrice > highestPrice * 0.9
drawdownFromHighest 数字 距最高价跌幅（%） drawdownFromHighest <= -20
profitPercent 数字 盈利百分比（%） profitPercent >= 30
holdDuration 数字 持仓时长（秒） holdDuration > 60
txVolumeU24h 数字 24小时交易量（USD） txVolumeU24h > 50000
holders 数字 持有者数量 holders >= 100
tvl 数字 总锁仓量（USD） tvl > 10000
fdv 数字 完全稀释估值（USD） fdv < 100000
marketCap 数字 市值（USD） marketCap > 5000

操作符

  • AND 或 &&: 且条件（两个条件都满足）
  • OR 或 ||: 或条件（任一条件满足）
  • > >= < <= == !=: 比较运算符

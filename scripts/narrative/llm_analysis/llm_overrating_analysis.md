# LLM 过度评分问题分析

## 发现的问题

### 1. 混淆"内容质量"与"代币叙事质量"

LLM容易被内容的"丰富性"迷惑，误判为高质量代币叙事：

| 代币 | LLM判断 | 实际问题 |
|------|---------|----------|
| 生菜 | "顶级谐音梗"传播力强 | 纯谐音，无代币价值主张 |
| 遗憾 | 情感叙事吸引力强 | 纯情感鸡汤，无代币实质 |
| 火鸡面 | 热搜事件有传播力 | 纯事件搬运，无代币关联 |
| 佛系 | 生活态度有创意 | 伪叙事，无代币用途 |

### 2. "伪叙事"模式未被识别

以下模式应该直接low，但LLM给high/mid：

1. **纯谐音梗**：生菜=生财、Duck you=鸭你一拳
2. **热搜搬运**：火鸡面、遗憾、再不吃就老了
3. **伪关联**：青蛙（童年故事提到一次）、佛系（生活态度）
4. **情感煽动**：遗憾的哲学叙事、社恐人情感共鸣

### 3. 可信度权重不足

即使缺乏权威来源，只要"有创意"+"传播力强"就得high：
- relink (86分): 声称"脑机接口医疗器械获批"，但来源只是推文
- MEME (65分): 声称"商标失效"，无权威验证

### 4. 评分阈值过低

当前阈值：
- high: ≥60分且内容≥20
- mid: 40-59分

问题：60分很容易达到（30分内容+20分传播+10分其他=60分）

## 改进方案 V4

### 核心原则：**区分"内容质量"与"代币叙事质量"**

### 1. 增加"代币实质"判断维度

内容必须回答以下问题之一才能获得高分：
- 代币的用途/功能是什么？
- 代币的价值主张是什么？
- 代币与特定场景/项目的关联是什么？
- 持有者为什么需要这个代币？

### 2. 明确"伪叙事"检测

以下情况直接返回low：
- **纯谐音梗**：只有谐音关联，无代币实质
- **热搜搬运**：纯报道热点事件，无代币意义说明
- **伪关联**：代币名称只在内容中被顺便提及
- **纯情感内容**：只有情感共鸣，无代币价值主张
- **虚假权威**：声称官方关联但无可靠验证

### 3. 提高可信度权重

- 没有权威来源的，最高只能是mid
- 声称"官方"、"币安"等但无可靠验证的，直接low
- 只有个人推特作为来源的，可信度≤10分

### 4. 调整评分权重

```
原权重：
- 内容质量: 35分
- 可信度: 30分
- 传播力: 20分
- 完整性: 15分

新权重：
- 可信度: 40分（提高权重）
- 代币实质: 35分（替换"内容质量"）
- 传播力: 15分（降低权重）
- 完整性: 10分
```

### 5. 提高high/mid阈值

```
- high: 总分≥75（原60）且 可信度≥30 且 代币实质≥25
- mid: 总分≥55（原40）且 可信度≥15
- low: 不满足以上条件
```
